# Texture Diffusion

Plugin Blender pour texturer des modèles 3D à partir d'images générées par IA, directement dans le viewport.

> 🖼️ **[SÉRIE 3 — (1) viewport 3D capturé / (2) image générée par l'IA / (3) résultat reprojeté sur le modèle]**

**Capture → IA → reprojection.** Tout ce que fait le plugin tient dans ces trois étapes. Cette doc explique ce qui se passe à chacune et ce que tu peux régler.

> 🖼️ **[FIGURE — sidebar 🍌 Texture Diffusion ouverte avec ses 4 panels visibles]**

Avant de commencer, colle ta clé API dans le panel **🔑 API Keys** (Replicate, fal.ai ou Gemini — voir [Modèles & tips](modeles.md) pour savoir laquelle choisir).

---

## 🎨 [Live Paint — retouches rapides](live-paint.md)

> 🖼️ **[FIGURE — capture du panel Generate avec son sub-panel Live Paint]**

Capture une zone du viewport, l'IA régénère, tu peins le résultat avec un stencil. Idéal pour boucher un trou ou corriger une zone sur une texture existante.

## 🌐 [Global Texture — texturer de A à Z](global-texture.md)

> 🖼️ **[FIGURE — capture du panel Global Texture avec ses 5 étapes visibles]**

Plusieurs vues générées sous différents angles, fondues avec des masques, bakées en une seule texture. Pour partir d'un mesh nu et obtenir une texture complète sur tout l'objet.

## 🧠 [Modèles & tips](modeles.md)

> 🖼️ **[FIGURE — galerie : un même prompt rendu par 4 modèles différents côte-à-côte]**

Comparatif des modèles disponibles (rapidité, coût, type de rendu), exemples côte-à-côte, et conseils pour optimiser tes générations. Le panel **💰 Cost** permet de suivre tes dépenses par scène.
